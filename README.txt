BOKNINGAR VERSION 2

NYHETER
- Dynamiska bokningsobjekt: Christin kan lägga till fler flikar/objekt.
- Visningsnamn i stället för e-postadress.
- Modernare startsida och kalender.
- Tryck på en ledig dag för att boka.
- Tryck på en bokad dag för att se vem som bokat.
- Installationsbar webbapp med ikon.

PUBLICERING
1. Säkerhetskopiera gärna den nuvarande index.html-filen i GitHub.
2. Ladda upp alla filer i denna mapp till GitHub:
   index.html
   manifest.webmanifest
   sw.js
   icon.svg
   firestore.rules
3. Öppna Firebase > Firestore Database > Rules.
4. Ersätt reglerna med innehållet i firestore.rules och klicka Publish.
5. Vänta cirka en minut på Netlify och uppdatera appen.
6. Logga in som barabajen@gmail.com. Standardobjekten Bil, Gotland och Bäcköra skapas automatiskt.
7. Gå till Inställningar för att lägga till fler bokningsobjekt.
