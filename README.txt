UPPLADDNING
1. Öppna GitHub-repot.
2. Add file > Upload files.
3. Ladda upp index.html, firestore.rules, icon.svg, _headers och README.txt.
4. Commit changes.
5. Vänta på Netlify Published.
6. Kontrollera att Version 2.2 visas.
